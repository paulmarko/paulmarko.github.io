Project Goals

Create a simple and reusable foundation for a numeric presentation.

  • single page application in ReactJS
  • build modular components in with Material-UI React Components
  • 100% JQuery free
  • structure code so that it can be used in a variety of projects
  • demonstrate code with a credit card banking application
  • add, filter, search, and view history via charts
  • simle REST API to interact with the server and db
