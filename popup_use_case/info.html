Project Goals

Create a one-stop web application to build, manage, and track popup advertisements on any website.



View On Any Website

Popups would be shown on any webpage where the launch code was placed.



Project Outcome

Application was built with 4 main user components:

  • Campaign Manager: turn a group of popup ads on/off, collect form data, and view reporting statistics
  • Ad Manager: turn off a single popup ad, set delivery page position, delay timers, and custom display rules.
  • Ad Editor: "what you see is what you get" editor, drag text, images, forms, into place, set background images, drag popup corners to resize
  • Launch Code: place on line of javascript on any website page to view popups in campaign


Development Specifics

  • Full stack LAMP/LEMP
  • 600,000+ lines of code
  • Symfony 1.4 PHP MVC & Doctrine ORM
  • JQuery / JQueryUI for all animations, and drag-n-drop advertisement editor

  • 1x MySQL database - RackSpace.com
  • 1x NodeJS server with Redis DB running websocket push notifications - RackSpace.com
  • 1,500 Unit & Functional Tests - 96% code coverage
  • 3 server environments - Live - TestLive - Development
  • 11 months to launch - first paying client in 3 weeks
